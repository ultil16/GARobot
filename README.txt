GARobotについて
本プログラムは大学の講義における自由課題で制作したものです。
設計から一人で行いましたが、様々なUnity関係の記事や講義内の理論を参考にしています。

プログラム概要
本プログラムは、遺伝的アルゴリズムによる進化を用いて、物理エンジン内のロボットに適した遺伝子型を探索するというものです。
原理やファイル構成についてはGARobo.pptx及び遺伝的アルゴリズムによるロボットの動きの探索のプログラム.docxに載せたスライドで詳しく説明しています。

実行方法
Assets/Scenes/EvolvingGoishiRobo.unityをunityで実行後、再生。
